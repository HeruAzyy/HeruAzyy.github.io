About

Heru Azy Pangestu. Jakarta, 11 Maret 2001
Saya seorang yang menyukai seni, bahkan untuk membangun sebuah website saya akan mempertimbangkan nilai seni didalamnya. Agar saya dapat menciptakan pengalaman penggguna yang baik saya belajar kurang lebih selama 2 tahun untuk mencapai itu semua.

Saya berfokus untuk membangun sebuah website yang inovatif dan mudah diakses semua orang. Beberapa fundamental web programming saya pelajari agar dapat saya terapkan ke dalam website inovatif yang sedang saya bangun.

Saat tidak didepan layar komputer mungkin saya sedang menikmati secangkir espresso, mendengarkan musik favorit, atau sekedar berkendara menikmati udara malam hari kota Semarang.
